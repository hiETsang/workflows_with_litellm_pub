## Tool Introduction 
- Design Spells is a thoughtfully curated collection that celebrates micro-interactions, easter eggs, and other seemingly small yet magical design elements found across the web. It aims to inspire designers by showcasing how these delightful details can elevate a website's personality and user experience.

## Key Features 
- Curated gallery of unique design details from various websites 
- Highlights innovative animations, interactions, and delightful surprises 
- Categories like micro-interactions, skeuomorphic elements, beautiful buttons etc.
