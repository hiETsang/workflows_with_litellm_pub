## Basic Information
- Website: https://mkdirs.com/
- Pricing: One-time payment of $119 (50% discounted price)
- Platforms: Web

## Tool Introduction
- Mkdirs is a feature-rich, customizable, and well-documented boilerplate to build directory websites with listings, payments, authentication, blog, and more. It aims to help you launch profitable directories quickly without building everything from scratch.

## Key Features
- Complete boilerplate with Next.js, Sanity CMS, Stripe, Authentication, and other industry-standard tech
- Secure authentication with email/password, social logins, password reset
- Paid/free submission system integrated with Stripe
- Blog with rich-text support, categories, and authors
- Email templates for onboarding, notifications, and newsletter
- SEO optimization with metadata and auto-generated sitemap
- Customizable UI with multiple layouts and dark mode

## Use Cases
- Building curated listings or directories for tools, services, resources etc.
- Monetizing the directory via paid submissions, sponsorships, affiliations
- Creating a blog/community around a specific niche

## Getting Started Guide
1. Clone the repository and customize as per your needs
2. Deploy the project on a platform like Vercel
3. Manage content via the integrated Sanity CMS
