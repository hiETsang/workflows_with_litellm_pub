## Mobbin > Mobbin is a comprehensive design inspiration library with over 300,000 mobile app and web UI screenshots.

## Basic Information - Official Website: https://mobbin.com/ - Pricing: Free with paid plans available - Platforms: Web, Figma plugin

## Tool Introduction - Mobbin is an invaluable resource for UI/UX designers seeking real-world inspiration and design patterns. Its massive library showcases screenshots from over 1,000 iOS, Android, and web apps, with new content added weekly.

## Key Features - Extensive library of 405,800 screens, 81,700 flows from 1,150 apps - Advanced filters to quickly find specific UI patterns, elements, flows - Insights into design patterns, copywriting, user flows from top apps - Figma plugin to copy designs directly into Figma projects

## Suitable Use Cases - Finding inspiration for mobile app and website UI design - Researching standard design patterns for common UI elements - Observing complete user flows with transitions and animations - Shortening design process by referencing existing implementations
